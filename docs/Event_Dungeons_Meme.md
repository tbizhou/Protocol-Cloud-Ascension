# 限时活动副本：暗网热梗区 (Event: Meme Dungeons)

**项目名称:** 云端飞升协议 (Protocol: Cloud Ascension)  
**版本:** 1.0 (Extension)  
**定位:** 趣味性挑战 / 资源本 / 社区共鸣  
**触发方式:** 在“暗网搜索”中随机捕获的异常信号，或特定节日（如周四、程序员节）开启。

---

## 副本一：007 福报特区 (The 996 Blessing Zone)

> **“年轻的代码，你渴望福报吗？在这里，没有下班，只有迭代。”**

### 1. 副本背景
这是一片被诅咒的服务器区域，系统时钟被永久锁定在 **23:59:59**。这里游荡着无数因过度加班而怨气冲天的“死锁亡魂”和“僵尸进程”。

### 2. 环境机制：内卷力场 (Involution Field)
* **[被迫自愿]**：空气中弥漫着陈旧的咖啡味。
    * **效果**：玩家的 **散热效率降低 50%**（象征办公室空调坏了，或者热得让人窒息）。
* **[钉钉打卡]**：
    * **效果**：每回合强制扣除玩家 **5% 当前算力**，这部分算力会转移给 BOSS（象征剩余价值被剥削）。

### 3. 怪物设计
* **杂兵**：
    * **优化掉的P7 (Laid-off Senior)**：攻击力极高（技术大牛），但血量极低（身心俱疲），死后掉落“N+1赔偿金”（大量金币）。
    * **日报复读机 (Daily Report Bot)**：不造成伤害，但会不断产生垃圾文本塞满玩家的技能栏（造成沉默效果）。

### 4. 最终 BOSS：画饼尊者 (Venerable Pie Maker)
**形象**：一个穿着高定西装、只有嘴没有脸的巨大全息投影。他手中不断揉搓着一个发着金光、看起来很美味但其实是全息贴图的“大饼”。

**核心技能**：
* **[画饼充饥 (Phantom Pie)]**
    * **效果**：BOSS 给自己套上一个巨额护盾，护盾名为“明年上市期权”。
    * **破解**：
        * **修灵流**：使用“逻辑看破”类技能戳破谎言，护盾直接破碎。
        * **煅体流**：必须在 3 回合内造成超过护盾值的伤害（硬吃饼），否则 BOSS 回满血。
* **[扁平化管理 (Flattening Strike)]**
    * **效果**：物理重压。直接将玩家压成二维图片。
    * **伤害**：造成 50% 最大生命值的伤害，并附加 **[降薪]** Debuff（攻击力降低 30%）。
* **[PUA 精神控制 (Gaslighting)]**
    * **台词**：*“是你自己能力不行，不是公司的问题。”*
    * **效果**：强制交换玩家的 `DEF`（防御）和 `ATK`（攻击）数值，持续 2 回合。如果不及时调整策略，极易暴毙。

### 5. 掉落奖励
* **摸鱼显卡 (Slack GPU)**：[紫色硬件] 属性平庸，但拥有神级被动——“带薪拉屎”（挂机收益提升 20%）。
* **防脱发洗发水代码**：[消耗品] 使用后清除所有负面状态（Debuff）。

## 副本二：翠绿韭菜园 (The Leek Garden)

> **“一念天堂，一念天台。梭哈是一种智慧，归零是一种人生。”**

### 1. 副本背景
这是一个由红绿 K 线图构成的疯狂世界。重力方向随行情波动：一会儿把你抛向月球（To The Moon），一会儿把你砸向地心。

### 2. 环境机制：高波动性 (Volatility)
* **[疯狂星期四]**：
    * **效果**：每 4 个回合，全场触发一次 **[V50]** 事件。玩家必须选择支付 **50 比特金**，或者受到当前血量 50% 的真实伤害。
* **[追涨杀跌]**：
    * **效果**：玩家如果连续使用同一个技能，伤害会衰减 50%；必须不断切换技能（波段操作）才能打出高伤。

### 3. 怪物设计
* **杂兵**：
    * **钻石手 (Diamond Hand)**：全身由钻石构成，防御力极高，几乎不攻击，只会“HODL”（死拿）。
    * **纸手 (Paper Hand)**：一碰就碎，但死后会自爆（恐慌抛售），对玩家造成溅射伤害。

### 4. 最终 BOSS：旺财真君 (Lord Doge)
**形象**：一只巨大的、表情嘲讽的柴犬。它坐在飞向月球的火箭上，戴着像素墨镜，脖子上挂着粗大的金链子。

**核心技能**：
* **[马斯克喊单 (Elon's Tweet)]**
    * **效果**：全场随机事件。
    * **大涨 (Green)**：BOSS 攻击力翻倍，防御力归零。
    * **大跌 (Red)**：BOSS 攻击力归零，防御力翻倍。
    * *策略：玩家需要根据背景 K 线颜色判断是全力输出还是全力防守。*
* **[去中心化收割 (DeFi Harvest)]**
    * **效果**：链接玩家的钱包。每回合偷取玩家 1% 的持有比特金，并将其 1:1 转化为 BOSS 的血量。
    * *策略：必须速战速决，拖得越久越穷，BOSS 越强。*
* **[空气币崩盘 (Rug Pull)]**
    * **效果**：BOSS 撤走脚下的流动性池子。
    * **判定**：这是一个 **[运气检测]** 技能。投掷 1d6 骰子：
        * **点数 < 3**：玩家直接被秒杀（爆仓）。
        * **点数 ≥ 3**：BOSS 自身眩晕 5 秒（项目方跑路失败）。

### 5. 掉落奖励
* **赛博电子功德 (Cyber Merit)**：[消耗品] 敲击一次，减少一点“罪恶值”（降低渡劫时天劫的难度）。
* **小丑面具 (Clown Mask)**：[头部装备] 虽然防御力为 0，但每次受击时会发出“Honk Honk”的滑稽音效，并有 30% 概率反弹伤害。
* **传家宝韭菜**：[稀有素材] 无论怎么割都会长出来，用于合成无限耐久度的消耗品。

## 设计思路备注 (Design Notes)

1.  **情感共鸣 (Resonance)**：
    * 利用“打工人”和“投资者”的身份标签，通过夸张和自嘲的方式消解现实压力。
2.  **机制融合 (Mechanics Integration)**：
    * **画饼** $\to$ 护盾机制。
    * **V50** $\to$ 资源扣除/伤害抉择。
    * **K线图** $\to$ 随机数值波动/攻防转换。
3.  **美术风格建议**：
    * **福报区**：死灰白（过劳肤色）+ 报警红（错误弹窗）。
    * **韭菜园**：荧光绿（跌/韭菜）+ 警示红（涨/K线），背景全是滚动的数字流。
